Light Emoting Diode



C#/.NET console app that uses facial recognition to determine expressions captured from a webcam. Commands are sent to an Arduino circuit based on the emotion detected to change the color of an LED light strip and RGB peripherals (keyboard, mouse, and headset).

Technology: C#, .NET, C++, Arduino

ContriBallad



Javascript web page which obtains a Github user contribution chart and uses it as a musical step sequencer. Each day of the week corresponds to an instrument, and the note or sound played is determined by the depth of contribution for each day. The scale and key are determined by the first letter of the username.

Technology: Javascript, Node.js, HTML, CSS
Live Site
