About Me

My name is Drew Byerly. I've been doing web design and development as a hobby since I was 11 years old. I started with free site builders like GeoCities and Freewebs, but quickly taught myself HTML, CSS, JavaScript, and eventually PHP. I studied Mechanical Engineering at Bradley University, and in 2015 I started an on-demand delivery service called All Things Delivered. While building the website and tools for the business, I realized I needed to pursue it as a career. I attended a full-time .net development program at Coding Temple in 2017, where I learned how to use a modern development stack, the .Net framework, and an object oriented programming languange (C#).
